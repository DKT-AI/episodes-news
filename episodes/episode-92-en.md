### 4. Trivy vulnerability scanner breach pushed infostealer via GitHub Actions
**Бриф:** Злоумышленники использовали компрометированные учетные данные для внедрения вредоносного ПО через официальные релизы и GitHub Actions сканера уязвимостей Trivy. Это позволило им собрать конфиденциальные данные, включая ключи SSH и облачные учетные данные, и обеспечить постоянный доступ к системам через systemd-сервис.
**Link:** https://www.bleepingcomputer.com/news/security/trivy-vulnerability-scanner-breach-pushed-infostealer-via-github-actions/
**Talking Points:**
- Как компрометация популярного инструмента безопасности, такого как Trivy, влияет на доверие к инструментам разработки и безопасности?
